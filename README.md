# Linux Process Manager

The following is a repository containing code for a Process manager built in C during my Operating Systems class.
This was created in order to familiarize myself with linux system calls such as fork() and exec*() as well as to familiarize myself with the idea of processes and background processes in general. It also consists of a linked-list implementation
to handle process control blocks and involves file parsing of /proc to retrieve required data for the process control blocks.

## How to run the process manager

Type make pman to build the project.
Then run the executable as ./pman
The prompt PMan > should then appear where you can start typing the required commands.
Type clean in order to clean the working environment.
